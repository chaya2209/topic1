IoT Applications

It is a platform which connects various devices(having various sensors) to the internet so that they can interact with each other. It is basically used for analyzing the data.

List of IoT Applications

  1. Air Quality Management System
  2. Agriculture
  3. Smart home security Systems
  4. Traffic Monitoring

Smart Buildings

Smart means to be interactive and responsive. Temperature/climate controlled building, smart officies are some examples of Smart building Scenarios

Some Features of Smart Buildings

  1. Helps Users to schedule according to their needs
  2. Life Safety
  3. Remote Access

Control Interfaces

The main parts of control interfaces are-:IoT Sensors, Analystics Software, User Interface and connectivity. Visitors, residents and utility providers etc are the users of this interface.

High-level Features

  1. Privacy
  2. Passive Design Techniques
  3. Energy Efficient
